# Concepts

CrucibleSignal defines reusable signal vocabulary without binding to a model family.

## What This Covers

Signals describe what was captured, not how a backend captured it.

## Worked Example

```elixir
CrucibleSignal.signal_types()
CrucibleSignal.capture_modes()
```
