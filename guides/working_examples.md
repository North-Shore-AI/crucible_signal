# Working Examples

The examples demonstrate mock and live-friendly signal handling.

## What This Covers

Mock examples run without model downloads. Live examples use Nx tensors and may skip cleanly if a backend is unavailable.

## Worked Example

```bash
mix run examples/signal_factory_mock.exs
mix run examples/tensor_summary_live.exs
```
